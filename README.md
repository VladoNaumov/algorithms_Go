Массивы (Arrays):

- 1,Максимальная прибыль (одна сделка),"Дан массив цен акций. Найти максимальную прибыль, которую можно получить от одной покупки и одной продажи."
- 2,Анализ баланса (Кадане),Дан массив ежедневных изменений баланса счёта. Найти максимальную сумму дохода за любой непрерывный период времени (накопленный прирост).
- 3,Сортировка по категориям (Dutch Flag),"Отсортировать массив заказов, содержащий только три приоритета: ""Низкий (0)"", ""Средний (1)"" и ""Высокий (2)"", чтобы сначала шли высокие, потом средние, потом низкие."
- 4,Сдвиг данных (Ротация логов),"Повернуть массив логов на K позиций вправо, имитируя ротацию старых данных."
- 5,Пропущенный платёж,"В списке из N ожидаемых платежей в диапазоне [1,N] найти единственный пропущенный платёж."
- 6,Единственный сотрудник,"В списке учёта рабочего времени, где ID каждого сотрудника записан дважды (вход и выход), кроме одного (забыл отметиться), найти ID этого сотрудника."
- 7,Продукт конкурентов,"Создать массив, где каждый элемент i — это произведение продаж всех конкурентов, кроме компании i. (Без использования деления)."
- 8,Максимальная последовательная убыль цен,"Дан массив цен акций. Найти максимальное количество последовательных дней, в течение которых цена падала."
- 9,Сдвиг неактивных пользователей,"Переместить все записи о неактивных пользователях (значение 0) в конец массива, сохраняя порядок активных пользователей."
- 10,Рейтинг популярности товара,Дан массив с оценками популярности (от 1 до 5) товаров. Найти 3 самых часто встречающихся (модальных) оценки.
- 11,Баланс транзакций,"Найти день, когда накопленный баланс был минимальным за весь период, используя массив ежедневных изменений."
- 12,Проверка контрольной суммы (ISBN),"Проверить, является ли контрольная сумма (взвешенная сумма) номера ISBN-10 корректной для проверки подлинности товара."
- 13,Поиск пиковой нагрузки,"Найти пиковый элемент в массиве потребления энергии, то есть момент, когда потребление выше, чем у ближайших соседних моментов."
- 14,Самая длинная серия побед/поражений,"Найти длину самой длинной последовательности подряд идущих одинаковых исходов (например, только 'W' или только 'L') в массиве результатов."
- 15,Проверка уникальности ID,"Проверить, содержит ли массив ID клиентов какие-либо дубликаты."

Два указателя (Two Pointers):

- 1,Целевая сумма инвестиций,"В отсортированном списке доходностей найти две доходности, сумма которых максимально близка к целевому годовому доходу."
- 2,Квадраты отсортированных значений,"Дан массив изменений цен (могут быть отрицательными). Вернуть массив квадратов этих изменений, отсортированный по возрастанию (для анализа волатильности)."
- 3,Удаление дубликатов (Склад),"Дан отсортированный список инвентарных номеров. Удалить дубликаты на месте, чтобы каждый номер остался только один раз."
- 4,Захват воды (Счётчик),"Дан массив высот стендов на конвейере. Вычислить, сколько жидкости может быть захвачено между стендами (например, при дожде)."
- 5,Сортировка чёт/нечёт (Чётность ID),"Разбить список ID заказов так, чтобы сначала шли чётные ID, а потом нечётные, сохраняя относительный порядок внутри групп."
- 6,Сумма трёх компонентов,"Найти все уникальные тройки компонентов в массиве, суммарный вес которых равен целевому весу партии (ноль, если ищем идеальное равновесие)."
- 7,Обратный порядок регионов,"Развернуть только гласные символы (или определённые кодовые буквы) в строке названия региона, не меняя остальные."
- 8,Удаление старых версий,"Удалить все вхождения заданного значения (например, устаревшей версии ПО) из массива на месте."
- 9,Сжатие данных,"Сжать строку повторяющихся символов (например, в серийном номере) в формат ""счётчик + символ"" (""AAAAB"" $\rightarrow$ ""4A1B"")."
- 10,Поиск медианы двух отсортированных баз данных,Найти медиану двух отсортированных массивов данных без фактического объединения баз.
- 11,Парковка (Two-Side Matching),"Найти максимальное количество автомобилей, которые можно поставить на парковку, используя отсортированные списки размеров мест и автомобилей."
- 12,Оптимизация долга/актива,"Найти две пары (Должник, Кредитор) с минимальной разницей, которые могут сбалансировать друг друга в отсортированных списках долгов и активов."
- 13,Корректировка палиндрома,"Проверить, является ли строка названия продукта палиндромом, игнорируя пробелы и пунктуацию."
- 14,Нахождение пары с близкой суммой,"Найти пару в отсортированном массиве значений, сумма которых наиболее близка к целевому значению."
- 15,Разделение положительных/отрицательных значений,"Переставить массив финансовых изменений так, чтобы все отрицательные числа (убытки) шли перед положительными (прибылями)."

Строки (Strings):

- 1,Разворот слов в адресе,Развернуть порядок слов в строке почтового адреса.
- 2,Проверка анаграмм (Кодовые слова),"Проверить, являются ли две строки кодовых слов анаграммами друг друга."
- 3,Самый длинный общий префикс (Файлы),Найти самую длинную общую часть (префикс) в массиве имён файлов или URL.
- 4,Проверка скобок (Код/Конфиг),"Проверить, правильно ли расставлены и закрыты скобки ((), [], {}) в строке конфигурационного файла или кода."
- 5,Преобразование в целое число (Парсинг),"Реализовать функцию, которая преобразует строку (например, введённую сумму) в целое число, обрабатывая знаки и переполнение."
- 6,Проверка изоморфных SKU,"Определить, являются ли два SKU (артикула) изоморфными (одинаковая структура, разная кодировка)."
- 7,Шифр Цезаря (Rot13),"Зашифровать/расшифровать строку сообщения, используя простой шифр Цезаря (используется в некоторых простых форумах)."
- 8,Самый часто повторяющийся символ (Статистика),Найти самый часто встречающийся символ в тексте документа.
- 9,Проверка на палиндром-перестановку,"Проверить, можно ли строку имени пользователя переставить, чтобы она стала палиндромом (для создания безопасного пароля)."
- 10,Сокращение пути (Unix-стиль),"Упростить заданный путь к файловой системе (например, /a/./b/../c/ → /c)."
- 11,Преобразование Excel столбца,"Вернуть номер столбца Excel (1, 2, 3...) по его буквенному названию (""A"", ""AB"", ""ZY"")."
- 12,Валидация IP-адреса,"Проверить, является ли данная строка корректным IP-адресом (IPv4)."
- 13,Парсинг CSV,"Реализовать базовый парсер CSV-строки, который разбивает строку на поля, учитывая кавычки."
- 14,Форматирование валюты,"Сформировать строку, представляющую валюту, добавив разделители тысяч и знак валюты."
- 15,Кодирование URL,Заменить пробелы в строке поискового запроса на последовательность %20 для корректного URL.

Скользящее окно (Sliding Window):

- 1,Длиннейшая подстрока без повторяющихся символов,"Найти длину самой длинной части ключа лицензии или ID транзакции, в которой все символы уникальны."
- 2,Минимальное окно-подстрока (Поиск тегов),"Найти кратчайшую подстроку в большом тексте, которая содержит все ключевые слова (теги) из списка."
- 3,Подмассив с заданной суммой (Целевой бюджет),"Найти непрерывный подмассив (окно) расходов, сумма которого точно равна заданному целевому бюджету."
- 4,Начало анаграмм (Анализ текстов),"Найти все стартовые индексы в большом тексте, где начинается анаграмма короткого кодового слова."
- 5,Максимальный средний балл (Прогноз),Найти непрерывный период (окно) длиной K дней с максимальным средним значением цены акции.
- 6,Максимум в скользящем окне,"Для каждого скользящего периода K дней, найти максимальное значение температуры или нагрузки на сервер."
- 7,Максимальное число последовательных периодов,"Дан бинарный массив (0/1). Найти длину самого длинного периода (окна), состоящего из единиц, если можно изменить не более K нулей (исправить K ошибок)."
- 8,Самая короткая подстрока с K уникальными символами,"Найти длину самой короткой подстроки, содержащей ровно K уникальных категорий товаров."
- 9,Подмассив с произведением меньше K,"Найти количество непрерывных подмассивов производительности, произведение которых меньше целевого порога K."
- 10,Минимальная длина подмассива (сумма ≥S),"Найти минимальную длину непрерывного подмассива заданий, сумма весов которых ≥ минимальный вес S."
- 11,Средняя температура за K дней,"Рассчитать массив, содержащий среднюю температуру за каждый скользящий период в K дней (для климатического анализа)."
- 12,Максимальное число клиентов в смене,"Найти максимальное количество клиентов, которые находились в магазине одновременно (окно)."
- 13,Максимальная прибыль с двумя транзакциями,"Найти максимальную прибыль, совершив две сделки (покупка/продажа) акций, без их пересечения."
- 14,Минимальное количество товаров для набора,"Найти минимальное количество последовательных товаров (окно), чтобы получить хотя бы один экземпляр каждого из K уникальных необходимых ингредиентов."
- 15,Длиннейшая последовательность покупок,"Найти самую длинную последовательность покупок, в которой уникальные товары встречаются не более K раз."

Матрицы (Matrices): 

- 1,Спиральный обход склада,"Вернуть все элементы (инвентарные номера) склада по спирали, имитируя инвентаризацию."
- 2,Поворот изображения (Сжатие),"Повернуть квадратную матрицу пикселей (изображение) на 90 градусов по часовой стрелке, используя минимум памяти."
- 3,Поиск в 2D-матрице (Отсортированная база),"Найти целевое значение (например, ID транзакции) в матрице, где строки и столбцы отсортированы."
- 4,Установка нулей (Распространение ошибки),"Если в ячейке производственного процесса обнаружен сбой (0), то весь ряд и столбец должны быть отмечены как сбойные (0)."
- 5,Самый длинный возрастающий путь (Топография),Найти длину самого длинного возрастающего маршрута в матрице высот местности.
- 6,Проход по диагоналям (Анализ данных),Вывести все элементы матрицы данных по диагоналям (для специфического анализа корреляций).
- 7,Число островов (Разбиение сети),"Дана двоичная матрица (1 — соединение, 0 — разрыв). Найти количество независимых сегментов сети (островов)."
- 8,Путь от угла до угла (Навигация),"Найти кратчайший путь в бинарной матрице (0 — препятствие, 1 — проход) от начальной точки до конечной."
- 9,Игра Жизни (Прогноз населения),"Применить правила ""Игры Жизни"" Конвея для моделирования роста/убыли популяции в сетке."
- 10,K-й наименьший элемент (Рейтинг),Найти K-й наименьший рейтинг в отсортированной по строкам и столбцам матрице рейтингов.
- 11,Поиск места в кинотеатре,Найти самую длинную последовательность свободных мест (0) в матрице плана кинотеатра.
- 12,Тепловая карта (Усреднение),"Создать новую матрицу, где каждое значение — это среднее показателей его соседей в окне 3×3."
- 13,Маршрут робота с минимальными затратами,Найти путь робота по сетке к цели с минимальной суммой логистических затрат (значений в ячейках).
- 14,Проверка судоку,"Проверить, является ли данная матрица 9×9 корректным (частичным) расписанием/планом, где уникальные элементы должны быть в каждой строке, столбце и подблоке."
- 15,Самая большая площадь прямоугольника из единиц,"Найти площадь самого большого прямоугольника, состоящего из свободных помещений (1) в плане здания (матрица)."

Интервалы (Intervals):

- 1,Слияние интервалов (Объединение встреч),Объединить все пересекающиеся интервалы занятости в календаре для получения сводного расписания.
- 2,Вставка интервала (Новая задача),Вставить новый интервал задачи в существующий отсортированный список и объединить его с любыми пересекающимися.
- 3,Непересекающиеся интервалы (Оптимизация),"Найти минимальное количество задач, которые нужно удалить, чтобы оставшиеся задачи не пересекались по времени."
- 4,Пересечение интервалов (Общее время),Найти пересечение (общий интервал) двух списков графиков работы (когда оба человека доступны).
- 5,Интервалы планировщика (Ресурсы),"Найти минимальное количество переговорных комнат, необходимых для проведения всех запланированных встреч."
- 6,Проверка на пересечение,"Определить, пересекаются ли два заданных срока выполнения проекта."
- 7,Пересечение двух списков интервалов,"Найти все временные интервалы, когда оба ресурса (например, Сервер 1 и Сервер 2) были активны."
- 8,Минимальное количество стрел (Кампании),"Найти минимальное количество рекламных кампаний (стрел), чтобы покрыть все запланированные временные интервалы."
- 9,Добавление интервалов (Начисление),"Применить ""добавки"" (например, бонусы) к соответствующим временным интервалам в журнале активности."
- 10,Перезагрузка интервалов (Свободное время),"Вернуть интервалы, когда человек доступен (не занят ни в рабочем расписании, ни в личных перерывах)."
- 11,Конфликты в расписании,"Вернуть все временные интервалы, когда оба человека (например, менеджер и клиент) свободны для встречи."
- 12,Планирование встреч,"Определить, возможно ли запланировать новую встречу заданной длительности D в свободное время в календаре."
- 13,Загруженность сервера,Найти максимальное количество одновременно подключенных пользователей (максимальное пересечение интервалов).
- 14,Календарь занятости (Summary),"Вернуть минимальный набор непересекающихся интервалов, полностью покрывающих всё время занятости."
- 15,Конфликты в бронировании,"Проверить, есть ли временное пересечение (конфликт) в массиве бронирований для одной переговорной комнаты."
